Sample:
SingleQuery

Description:
This sample shows how to do queries on a single category and how to process the result data.
The source code is taken from the API Documentation as followed:

Method				API Documentation Chapter
btn_simple_Click	Developers Guide->Queries->Find Documents->Simple Query
btn_execute_Click	Developers Guide->Queries->Find Documents->Query in One Category
btn_process_Click	Developers Guide->Queries->Find Documents->Process Query Results

Requirements:
* The category "Files" must have been importet or the category name in line 63 has to be changed as well.
* You need to have at least one document in this category.

Build-Steps:
* If you change the category name you may also have to change the field names in lines 75 & 78.
