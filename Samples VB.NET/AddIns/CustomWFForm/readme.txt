Sample:
CustomWFForm

Description:
This sample shows how to display a self-created dialog for workflow instances of a specific category and a specific task.
In addition it is shown how to end a workflow task and proceed with the next task.

Requirements:
* Make sure that the task name exists in your workflow for the used category.

Installing:
* For information about installing Therefore AddIns take a look at Developers Guide->AddIns->AddIn Creation and Configuration
  in the Therefore API document.
