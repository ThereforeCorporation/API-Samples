Sample:
DocumentOperations

Description:
This sample shows how to retrieve a document, extract files from the document and how to add a 
file to a Therefore document.

Therefore™ Web service operations:
- GetDocument
- UpdateDocument

Requirements:
* Assure that the "Files" category has been imported into your Therefore system.
* You need at least one valid document in your Therefore system
* Make sure the category and field numbers in the file Common.CategoryIds.cs are valid.
