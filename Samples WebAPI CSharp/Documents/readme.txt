Sample:
Documents

Description:
This sample shows how to do the most common operations regarding to the Therefore™ documents. This
includes archiving, retrieving a document as well as extracting files from the document

Therefore™ Web service operations:
- CreateDocument
- GetDocument


Requirements:
* Assure that the "Files" category has been imported when trying this sample.
* Make sure the category and field numbers in the file Common.CategoryIds.cs are valid.
