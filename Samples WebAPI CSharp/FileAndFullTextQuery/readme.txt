Sample:
FileAndFullTextQuery

Description:
This sample shows two things. First is how to do a file query and the second one is how to do a
full-text query.

Therefore™ Web service operations:
- ExecuteFullTextQuery
- ExecuteSingleQuery


Requirements:
* The Table created by the DataType.sql file hast to exist in your Therefore Database
* Assure that you have importet the "Files" and "Simple Invoice" categories when trying this sample.
* You need to have at least one document in each category.
* Make sure the category and field numbers in the file Common.CategoryIds.cs are valid.
