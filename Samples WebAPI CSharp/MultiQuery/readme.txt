Sample:
MultiQuery

Description:
This sample shows how to do queries on multiple categories and how to process the result data.

Therefore™ Web service operations:
- ExecuteMultiQuery


Requirements:
* The Table created by the DataType.sql file has to exist in your Therefore Database
* You need to have at least one document in each category.
* Make sure the category and field numbers in the file Common.CategoryIds.cs are valid.
