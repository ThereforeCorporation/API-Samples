Sample:
SingleQuery

Description:
This sample shows how to do queries on a single category and how to process the result data.

Therefore™ Web service operations:
- ExecuteSingleQuery
- GetQuerySpecification

Requirements:
* The category "Files" must have been imported.
* You need to have at least one document in this category.
* Make sure the category and field numbers in the file Common.CategoryIds.cs are valid.
